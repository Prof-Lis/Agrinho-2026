Portal de Desenvolvimento Sustentável: Integração Produção e Preservação

Este repositório contém uma aplicação web interativa desenvolvida com foco pedagógico para explorar e ensinar a integração entre práticas de produção industrial/agrícola e a preservação ambiental, apoiando-se em inovações tecnológicas contemporâneas.

📌 Origem do Projeto e Créditos

Este projeto e toda a fundamentação pedagógica e tecnológica contida no portal foram desenvolvidos a partir da pesquisa acadêmica e conceitual solicitada por Lis Gugelmin.

O conteúdo baseia-se no estudo de paradigmas de sustentabilidade que buscam harmonizar a eficiência de mercado com a resiliência dos sistemas biofísicos terrestres, aplicando esses conceitos de forma prática ao ensino de robótica e computação.

🚀 Funcionalidades do Portal

O portal sintetiza as principais discussões globais sobre desenvolvimento sustentável e tecnologia por meio de uma interface visualmente rica e interativa:

Painel Multicritério das Dimensões da Sustentabilidade: Apresenta visualmente as seis dimensões essenciais discutidas no plano (Ambiental, Socioeconômica, Governança, Agro/Industrial, Cultural e Biodiversidade).

Gráfico Interativo (Linear vs. Circular): Um gráfico de barras dinâmico (desenvolvido com Chart.js) que compara de forma didática o ciclo de vida dos produtos nos modelos de Economia Linear e Economia Circular.

Barra de Busca Inteligente: Permite filtrar os cartões de conteúdo em tempo real, facilitando a navegação de estudantes e professores entre temas como IA Verde, Indústria 5.0, Blockchain e a Educação no Paraná.

Design Responsivo & Temático: Estética moderna utilizando tons de verde e azul (representando a fusão de preservação e tecnologia) totalmente adaptável para dispositivos móveis e desktops através do Tailwind CSS.

🛠️ Tecnologias Utilizadas

O portal foi construído como uma aplicação de página única (Single Page Application), otimizando a distribuição e o carregamento rápido:

HTML5 & Javascript (ES6+) para estruturação e interatividade.

Tailwind CSS para estilização utilitária rápida e layout responsivo.

Chart.js para renderização dinâmica de dados estatísticos de resíduos.

💻 Como Executar o Projeto

Como o portal foi projetado sob a filosofia de arquivo único, não é necessária nenhuma etapa complexa de compilação ou instalação de dependências locais:

Faça o clone deste repositório:

git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)


Navegue até a pasta do projeto:

cd seu-repositorio


Abra o arquivo sustentabilidade_pedagogica.html diretamente em qualquer navegador de sua preferência.
